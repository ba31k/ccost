# ccost

Считает, сколько токенов и денег (по прайсу API) потрачено во всех локальных сессиях Claude Code.

Один файл, чистый Python 3, без зависимостей. Читает `~/.claude/projects/**/*.jsonl`,
дедуплицирует вызовы по `message.id` (иначе resume/fork сессий удваивают цифры) и считает
стоимость по стандартному прайсу API.

## Запуск

```sh
./ccost
```

или положить в `PATH`:

```sh
cp ccost ~/bin/          # если ~/bin в PATH
ccost
```

## Пример вывода

```
  Всего:  4,283,097,475 токенов   ≈ $3,094.28 (по прайсу API)

  модель                        input    cache   output           $
  ---------------------------------------------------------------
  claude-opus-4-6               89.4K     2.1B     8.2M    1,512.30
  claude-opus-4-8                1.2M     1.8B    10.1M    1,393.10
  claude-sonnet-4-6              7.5K   257.8M   378.6K       94.15
  ...
  ---------------------------------------------------------------
  ИТОГО                          1.4M     4.3B    19.3M    3,094.28
```

## Как считает

- **цены** — стандартный прайс API ($/1M токенов), см. таблицу `PRICE` в скрипте;
- **кэш** — запись ×1.25 (5-мин ephemeral), чтение ×0.1 от input-цены;
- `cache` в выводе — это `cache_write + cache_read` вместе.

> Это цена **по прайсу API**, а не то, что реально заплачено на подписке —
> ровно как показывает `/status`.

## Заметки

- Новые модели должны быть в таблице `PRICE`, иначе строка помечается `⚠ нет цены`.
- Множители кэша заданы под 5-минутный ephemeral-кэш Claude Code.
