# ccost

Статистика токенов и денег (по прайсу API) по всем локальным сессиям Claude Code —
интерактивный TUI прямо в терминале.

Один файл, чистый Python 3, без зависимостей (TUI на stdlib `curses`).
Читает `~/.claude/projects/**/*.jsonl`, дедуплицирует вызовы по `message.id`
(иначе resume/fork сессий удваивают цифры) и считает стоимость по прайсу API.

## Запуск

```sh
./ccost              # интерактивный TUI
./ccost report       # полный текстовый отчёт (он же фолбэк для пайпа / не-tty)
./ccost -r           # то же, что report
```

или положить в `PATH`:

```sh
cp ccost ~/bin/      # если ~/bin в PATH
ccost
```

## TUI

Вкладки: **Обзор · Модели · Дни · Чаты · Проекты · Часы**.

Управление:

| клавиша | действие |
|---|---|
| `←` `→` / `1`–`6` / `Tab` | переключение вкладок |
| `↑` `↓` / `j` `k` / `PgUp` `PgDn` | скролл |
| `f` / `F` | период: всё → сегодня → 7 дней → 30 дней |
| `s` | сортировка: по стоимости ↔ по токенам |
| `q` / `Esc` | выход |

- **Обзор** — общий итог, ключевые метрики (чаты, дни, пик дня/часа, доля кэша),
  разбивка токенов и спарклайн стоимости по дням.
- **Чаты** — топ разговоров по стоимости, с заголовком (первое сообщение) и моделью.
  Usage сабагентов сворачивается в родительский чат.
- **Дни / Часы** — распределение во времени с барами.

## Как считает

- **цены** — стандартный прайс API ($/1M токенов), см. таблицу `PRICE` в скрипте;
- **кэш** — запись ×1.25 (5-мин ephemeral), чтение ×0.1 от input-цены;
- `cache` в выводе — это `cache_write + cache_read` вместе.

> Это цена **по прайсу API**, а не то, что реально заплачено на подписке —
> ровно как показывает `/status`.

## Цвета

Палитра подстраивается под терминал: фон берётся родной (`use_default_colors`),
цвета выбираются по числу поддерживаемых (256 → аккуратные оттенки, 8/16 → базовые,
без цвета → только жирный/инверсия). Работает и на тёмном, и на светлом фоне.

Полностью выключить цвета: `NO_COLOR=1 ccost`.

## Заметки

- Новые модели должны быть в таблице `PRICE`, иначе стоимость по ним = $0.
- Множители кэша заданы под 5-минутный ephemeral-кэш Claude Code.
- Записи с нулём токенов (синтетические сообщения) отбрасываются.
